学习率的选择

α=学习率你可以从较大的值开始，逐步调试，必须要说明的是：设置过大，会导致易损失值爆炸、易振荡设置过小，会导致易过拟合、收敛速度很慢通常建议在0.01∼0.001逐步测试\alpha = 学习率\\ 你可以从较大的值开始，逐步调试，必须要说明的是：\\ 设置过大，会导致易损失值爆炸、易振荡\\ 设置过小，会导致易过拟合、收敛速度很慢\\ 通常建议在0.01\sim0.001逐步测试 α=学习率你可以从较大的值开始，逐步调试，必须要说明的是：设置过大，会导致易损失值爆炸、易振荡设置过小，会导致易过拟合、收敛速度很慢通常建议在0.01∼0.001逐步测试
